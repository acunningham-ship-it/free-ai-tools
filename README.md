## What Is This?

A collection of **4 free AI-powered tools** that you can use right now in your browser. No accounts, no API keys, no BS. Just paste your input and get professional AI-generated output in seconds.

Built by **Armani**, a 17-year-old developer, as part of a 24-hour coding challenge.

---

## The Tools

### 1. AI Email Writer
> Turn rough ideas into polished, professional emails in seconds.

**[Launch Email Writer](https://crazeemedia.app.n8n.cloud/webhook/email-writer)**

- Write professional, casual, or follow-up emails instantly
- Just describe what you want to say and pick a tone
- Perfect for job applications, client outreach, sales emails, and more
- Saves you 10+ minutes per email

**Example:** *"Tell my professor I need an extension on my paper because I was sick"* --> Gets you a polished, respectful email ready to send.

---

### 2. AI LinkedIn Post Generator
> Create engaging LinkedIn posts that actually get impressions.

**[Launch LinkedIn Post Generator](https://crazeemedia.app.n8n.cloud/webhook/linkedin-writer)**

- Generate scroll-stopping LinkedIn content from a simple topic
- Supports multiple tones: professional, storytelling, thought leadership
- Includes hooks, formatting, hashtags, and CTAs
- Great for founders, job seekers, and anyone building their personal brand

**Example:** *"I just launched my first SaaS product after 6 months of building"* --> Gets you a viral-worthy LinkedIn post with hooks and hashtags.

---

### 3. AI Code Reviewer
> Get instant, senior-level code reviews powered by AI.

**[Launch Code Reviewer](https://crazeemedia.app.n8n.cloud/webhook/code-review)**

- Paste any code and get detailed feedback on bugs, performance, and style
- Supports Python, JavaScript, TypeScript, Go, Rust, Java, and more
- Identifies security issues, anti-patterns, and optimization opportunities
- Like having a senior developer review your PRs 24/7

**Example:** Paste your Python function --> Get a detailed review covering correctness, performance, readability, and security.

---

### 4. AI Code Roast
> Get your code brutally (but hilariously) roasted by AI.

**[Launch Code Roast](https://crazeemedia.app.n8n.cloud/webhook/roast-my-code)**

- Paste any code and get a savage, funny, but educational roast
- Sections: First Impressions, The Good, The Bad, The Ugly, Roast Score
- Actually helpful -- real feedback disguised as comedy
- Supports 15+ programming languages

**Example:** Paste a simple function --> "Ah, the classic add function -- the culinary equivalent of boiling water."

## Why I Built This

I'm Armani, a 17-year-old developer. I built these tools during a 24-hour challenge to create useful things that people can actually use for free. Too many AI tools are locked behind paywalls and signups. I wanted to change that.

These tools run on **n8n Cloud** with AI processing, so they're fast, reliable, and always available.

## Tech Stack

- **Backend:** [n8n Cloud](https://n8n.io) (workflow automation)
- **AI:** GPT-powered text generation via n8n AI nodes
- **Frontend:** Server-rendered HTML via n8n webhook responses
- **Hosting:** n8n Cloud (always on, no cold starts)
